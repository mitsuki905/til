# AI開発の作法・進め方

## アプリ開発の流れ（例：ToDoアプリ）

### 1. 要件定義

- 不明点をつぶしていきたいので、わからないところをどんどん質問する
- 動作環境・使用言語などを明確にする
  - 例：Windows で動かす、Python で開発

### 2. 要件定義書の作成

AIに以下のように指示すると、要件定義書として整形してくれる：

```
要件定義書として保存したいので不要な文章は全部削除
最初と最後の挨拶は不要
純粋な要件定義書の文書として出力して
```

→ `requirements.md` というファイルを作成して、出力内容をコピペする

### 3. アプリの実装

- VSCode の **Cline**（AI拡張機能）を使って要件定義をもとにアプリを作成する
- ⚠️ **費用がかかるので注意**（APIの利用料金が発生する）
